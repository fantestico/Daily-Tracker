# 🌿 Daily Task Progress Tracker

A beautifully designed, "soft editorial" daily task management web application. It functions as an infinite calendar to track habits and daily tasks seamlessly across time, drawing inspiration from premium paper notebooks and planners.

**🌍 Live Website:** [https://fantestico.github.io/Daily-Tracker/](https://fantestico.github.io/Daily-Tracker/)

## ✨ Features
- **Infinite 10-Day Pages:** Scroll forward and backward infinitely through time, visualizing 10 days of progress at once.
- **Dynamic Progress Bars:** Automatically recalculates your total daily task completion rate and displays it via sleek CSS progress bars.
- **Progressive Web App (PWA) Ready:** Install the website directly to your iOS or Android home screen to use it as a standalone native app! It works offline too.
- **Local Persistence:** Your tasks and completion history are securely saved directly within your browser's local storage.
- **Soft Editorial Aesthetic:** Calm parchment background (`#F7F5F0`), deep charcoal typography (`DM Serif Display` & `DM Sans`), and sage green accents (`#5C7A5F`).

## 🛠️ Technology Stack
- **HTML5** & **Vanilla CSS3** (Flexbox/Grid layout, Custom Animations)
- **Vanilla JavaScript** (State Management, Local Storage, DOM Manipulation)
- **Service Workers & Manifest** (PWA support and offline caching)

## 📱 How to Install the Mobile App
1. Open the [live website](https://fantestico.github.io/Daily-Tracker/) on your phone's browser (Safari/Chrome).
2. Tap the **"Download App"** button at the top right of the screen.
3. Follow the prompt to add the application to your home screen!

## 🚀 Running Locally
If you want to run this project locally on your machine:
1. Clone the repository: `git clone https://github.com/fantestico/Daily-Tracker.git`
2. Navigate into the directory: `cd Daily-Tracker`
3. Serve it using a simple HTTP server, for example: `python -m http.server 8000`
4. Open your browser and go to `http://localhost:8000`
